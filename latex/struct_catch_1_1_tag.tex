\doxysubsection{Detailed Description}
A {\bfseries{view}} of a tag string that provides case insensitive comparisons

Note that in Catch2 internals, the square brackets around tags are not a part of tag\textquotesingle{}s representation, so e.\+g. \char`\"{}\mbox{[}cool-\/tag\mbox{]}\char`\"{} is represented as \char`\"{}cool-\/tag\char`\"{} internally.
