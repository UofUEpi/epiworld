Setting and accessing parameters from the model. 

Tools can incorporate parameters included in the model. Internally, parameters in the tool are stored as pointers to an std\+::map$<$$>$ of parameters in the model. Using the {\ttfamily unsigned int} method directly fetches the parameters in the order these were added to the tool. Accessing parameters via the {\ttfamily std\+::string} method involves searching the parameter directly in the std\+::map$<$$>$ member of the model (so it is not recommended.)

The function {\ttfamily set\+\_\+param()} can be used when the parameter already exists in the model.

The {\ttfamily par()} function members are aliases for {\ttfamily get\+\_\+param()}.
