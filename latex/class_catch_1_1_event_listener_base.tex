\doxysubsection{Detailed Description}
Base class to simplify implementing listeners.

Provides empty default implementation for all \mbox{\hyperlink{class_catch_1_1_i_event_listener}{IEvent\+Listener}} member functions, so that a listener implementation can pick which member functions it actually cares about.
